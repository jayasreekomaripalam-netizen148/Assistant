# 🤖 ECET Assistant Chatbot

An AI-powered chatbot built using Poe to help students prepare for the ECET exam with explanations, doubt solving, and quiz practice.

---

## 🚀 Features

* 📘 Answers any questions clearly and very shortly 
* 🧠 Provides step-by-step explanations
* ❓ Generates quiz questions
* ✅ Evaluates answers with explanations
* 🎯 Helps in exam preparation

## 📚 Usage

1. Open the chatbot link
2. Ask any ECET question
3. Type **"quiz"** to start practice
4. Answer questions and get feedback
